## 项目简介
本项目为课程学习的作业Demo，主要用于展示UE5第一人称射击游戏下的基础玩法逻辑及多人网络同步的实现。

## 功能描述

项目完整实现了从敌人移动逻辑、玩家战斗逻辑、得分判定到多人网络同步的基础游戏流程，详情如下：

- 实现了完整的 **敌人系统**。敌人以球体作为 Mesh 表现形式，基于 **NavMesh** 自动寻路，并主动追踪距离最近的玩家；当玩家与敌人距离过近时，玩家会持续受到伤害并扣除生命值；场景中设置同时最多存在 **两个敌人**，敌人被击败后会在场景中自动刷新新的敌人。
- 实现了**多人游戏模式**。项目采用 **Listen Server** 网络模式：一名玩家作为主机（Host）启动游戏并承担服务器职责，其他玩家以客户端方式加入同一游戏会话；游戏中的关键逻辑（敌人行为、伤害判定、得分与胜利状态）均由服务器进行权威判定，并同步到各客户端，保证多人游戏状态一致。
- 实现了 **玩家战斗与生命机制**。玩家初始携带枪械，可对敌人进行射击并造成伤害；玩家生命值降低至 0 后会在地图中重生，并可继续参与游戏。
- 设计并实现了 **得分与胜利条件机制**。在多人测试中，使用两名玩家进行了同时游戏测试；每击败一个敌人，全局得分加 1，当累计击败敌人数量达到 5 个时，触发游戏胜利显示界面。
- 实现并优化了基础 **HUD 与射击体验**。在屏幕中心添加十字准星，提升射击的准确性；实现 HUD 血条与得分显示，并在多人模式下保持同步更新。

## 开发环境

- Unreal Engine 版本：UE 5.2.1
- Visual Studio：2022
- 操作系统：Windows 10
- 开发模板：First Person Template

## 如何运行项目
1. 克隆仓库到本地
2. 使用 Unreal Engine 5 打开 `demo.uproject`
3. 等待工程加载完成后，点击 `Play` 即可运行
4. **视频演示见[链接](https://www.bilibili.com/video/BV1yQv6BkEEe/)**
